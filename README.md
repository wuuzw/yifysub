# yifysub

yifysub is a cli tool to download subtitles from [yifysubtitles.com](https://www.yifysubtitles.com/)

## Installation

Use the package manager [npm](https://nodejs.org/) to install yifysub.

```shell
npm install -g yifysub
```

## Usage

```shell
yifysub
```
